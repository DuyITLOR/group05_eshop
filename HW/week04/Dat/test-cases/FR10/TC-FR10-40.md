## Test steps
1. Gửi yêu cầu thay đổi trạng thái đơn hàng sang `pending` bằng cách gọi API Admin `PUT /api/admin/orders/:id/status` hoặc gọi API User hủy `PUT /api/orders/:id/cancel`.

## Expected results
1. Hệ thống từ chối yêu cầu, trả về HTTP status 401 Unauthorized hoặc 403 Forbidden do tài khoản User gọi API Admin.
2. Trạng thái đơn hàng trong DB giữ nguyên là `delivered`.

## Actual results
1. Từ chối yêu cầu, HTTP status 400.
2. Trạng thái đơn hàng trong DB giữ nguyên là `delivered`.
