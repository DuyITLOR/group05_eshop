## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái kết thúc `delivered` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Trạng thái hiện tại | delivered |
| Trạng thái cập nhật | shipping |

## Test steps
1. Tìm đơn hàng có trạng thái `delivered` trong danh sách đơn hàng.
2. Thử cập nhật trạng thái đơn hàng sang `shipping` bằng cách gọi API với `status = 'shipping'`.
3. Xác nhận thực hiện.

## Expected results
1. Hệ thống từ chối cập nhật do `delivered` là trạng thái kết thúc (Final State).
2. Trả về thông báo lỗi phù hợp (HTTP 400 Bad Request).
3. Trạng thái đơn hàng trong DB giữ nguyên là `delivered`.
