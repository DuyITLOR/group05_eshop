test-design/EP-FR10.md

## Detail
| ID | Test Objective | State Transition (Input) | Kết quả (Output) |
|---|---|---|---|
| FR10_EP_IV05 | Admin cập nhật sang trạng thái không xác định (`pending` -> `unknown`) | `pending` -> `unknown` (ST10) | Báo lỗi chuyển đổi không hợp lệ (O2) |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái `pending` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Trạng thái hiện tại | pending |
| Trạng thái cập nhật | unknown |

## Test steps
1. Tìm đơn hàng có trạng thái `pending` trong danh sách đơn hàng.
2. Thử gửi yêu cầu cập nhật trạng thái đơn hàng sang một giá trị không xác định/không hợp lệ (`unknown`).
3. Xác nhận thực hiện.

## Expected results
1. Hệ thống từ chối cập nhật trạng thái.
2. Trả về thông báo lỗi phù hợp (HTTP 400 Bad Request).
3. Trạng thái đơn hàng trong DB giữ nguyên là `pending`.

## Actual results
1. Thành công: Hệ thống từ chối cập nhật trạng thái đơn hàng.
2. Thành công: Trả về mã lỗi HTTP 400 Bad Request.
3. Thành công: Trạng thái đơn hàng giữ nguyên là `pending`.

## Status
Passed
