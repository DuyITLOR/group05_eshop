test-design/EP-FR10.md

## Detail
| ID | Test Objective | Token (Input) | Order ID (Input) | State Transition (Input) | Expected Output |
|---|---|---|---|---|---|
| FR10_EP_IV08 | Thực hiện cập nhật trạng thái khi chưa đăng nhập hoặc token sai | Token không hợp lệ (TOK_A3) | Đơn hàng tồn tại (ORD1) | pending -> confirmed (ST1) | - Báo lỗi chưa đăng nhập (O4)<br>- Trả về HTTP 401 Unauthorized |

## Preconditions
1. Tồn tại một đơn hàng ở trạng thái `pending` trong hệ thống.

## Test data
| Field | Value |
|---|---|
| Token JWT | Token không hợp lệ / không gửi token (TOK_A3) |
| Order ID | Đơn hàng tồn tại (ORD1) |
| Cặp chuyển đổi | pending -> confirmed (ST1) |

## Test steps
1. Thực hiện gửi yêu cầu cập nhật trạng thái đơn hàng sang `confirmed` qua API Admin mà không gửi kèm Token (hoặc Token sai chữ ký).

## Expected results
1. Hệ thống từ chối thực hiện do chưa xác thực.
2. Trả về mã lỗi HTTP 401 Unauthorized.
3. Trạng thái đơn hàng giữ nguyên là `pending`.

## Actual results
1. Thành công: Hệ thống từ chối cập nhật trạng thái và trả về mã lỗi HTTP 401 Unauthorized do token không hợp lệ/thiếu token.
2. Thành công: Trạng thái đơn hàng giữ nguyên là `pending`.

## Status
Passed
