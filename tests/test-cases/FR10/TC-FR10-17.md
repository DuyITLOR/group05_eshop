## Preconditions
1. Người dùng A đã đăng nhập hệ thống với tài khoản User.
2. Tồn tại một đơn hàng thuộc sở hữu của người dùng B (User B) ở trạng thái `pending`.

## Test data
| Field | Value |
|---|---|
| Token JWT | Token của User A (TOK_U1) |
| Order ID | Đơn hàng của User B (ORD_U2) |
| Trạng thái hiện tại | pending (CS_U1) |

## Test steps
1. User A thực hiện gửi yêu cầu hủy đơn hàng của User B bằng cách gọi API với Order ID của User B.

## Expected results
1. Hệ thống từ chối hủy đơn hàng và trả về mã lỗi không tìm thấy đơn hàng hoặc không có quyền (HTTP 404 Not Found hoặc HTTP 403 Forbidden).
2. Trạng thái đơn hàng của User B trong DB vẫn giữ nguyên là `pending`.

## Actual results
1. Thành công: Hệ thống từ chối hủy đơn hàng của người khác.
2. Thành công: Trả về mã lỗi HTTP 404 Not Found.
3. Thành công: Trạng thái đơn hàng của User B trong DB giữ nguyên là `pending`.
