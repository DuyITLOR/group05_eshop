## Detail
| ID | Test Objective | Token (Input) | Order ID (Input) | State Transition (Input) | Expected Output |
|---|---|---|---|---|---|
| FR10_EP_IV06 | Admin cập nhật sang trạng thái không xác định (`pending` -> `unknown`) | Admin Token (TOK_A1) | Đơn hàng tồn tại (ORD1) | pending -> unknown (ST10) | - Báo lỗi chuyển đổi không hợp lệ (O2)<br>- Trả về HTTP 400 Bad Request |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái `pending` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Token JWT | Admin Token hợp lệ (TOK_A1) |
| Order ID | Đơn hàng tồn tại (ORD1) |
| Cặp chuyển đổi | pending -> unknown (ST10) |

## Test steps
1. Tìm đơn hàng có trạng thái `pending`.
2. Thử gửi yêu cầu cập nhật trạng thái đơn hàng sang một giá trị không xác định (`unknown`).

## Expected results
1. Hệ thống từ chối cập nhật trạng thái.
2. Trả về mã lỗi HTTP 400 Bad Request.
3. Trạng thái đơn hàng trong DB giữ nguyên là `pending`.

## Actual results
1. Thành công: Hệ thống từ chối cập nhật trạng thái đơn hàng.
2. Thành công: Trả về mã lỗi HTTP 400 Bad Request.
3. Thành công: Trạng thái đơn hàng giữ nguyên là `pending`.
