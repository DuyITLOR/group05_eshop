## Detail
| ID | Test Objective | State Transition (Input) | Kết quả (Output) |
|---|---|---|---|
| FR10_EP_V03 | Admin cập nhật trạng thái đơn hàng `confirmed` -> `shipping` | `confirmed` -> `shipping` (ST3) | Cập nhật thành công, trạng thái mới là `shipping` (O1) |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái `confirmed` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Trạng thái hiện tại | confirmed |
| Trạng thái cập nhật | shipping |

## Test steps
1. Tìm đơn hàng có trạng thái `confirmed` trong danh sách đơn hàng.
2. Chọn đơn hàng đó và cập nhật trạng thái sang `shipping`.
3. Xác nhận lưu thay đổi.

## Expected results
1. Trạng thái đơn hàng được cập nhật thành công thành `shipping` trên giao diện Admin.
2. Trạng thái đơn hàng được cập nhật thành công thành `shipping` trong cơ sở dữ liệu.
3. Trả về phản hồi thành công (HTTP 200 OK).
