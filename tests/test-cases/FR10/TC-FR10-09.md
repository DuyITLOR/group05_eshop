## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái `pending` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Trạng thái hiện tại | pending |
| Trạng thái cập nhật | pending |

## Test steps
1. Tìm đơn hàng có trạng thái `pending` trong danh sách đơn hàng.
2. Thực hiện cập nhật trạng thái đơn hàng sang chính nó (`pending`) bằng cách gọi API với `status = 'pending'`.
3. Xác nhận thực hiện.

## Expected results
1. Hệ thống từ chối cập nhật trạng thái hoặc báo lỗi chuyển đổi trạng thái không hợp lệ.
2. Trả về lỗi phù hợp (HTTP 400 Bad Request).
3. Trạng thái đơn hàng giữ nguyên là `pending`.
