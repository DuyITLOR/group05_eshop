## Preconditions
1. Người dùng đã đăng nhập hệ thống với tài khoản User.
2. Tồn tại đơn hàng ở trạng thái kết thúc `delivered` do chính User này đặt.

| Field | Value |
|---|---|
| Token JWT | User Token hợp lệ (TOK_U1) |
| Order ID | Đơn hàng của chính mình (ORD_U1) |
| Trạng thái hiện tại | delivered (CS_U4) |

## Test steps
1. Kiểm tra giao diện xem có nút hủy đơn hàng cho đơn hàng đã giao thành công hay không.
2. Thử gửi yêu cầu hủy đơn hàng qua API hủy.

## Expected results
1. Giao diện người dùng không cung cấp nút hủy đơn hàng.
2. Hệ thống từ chối yêu cầu và trả về lỗi HTTP 400 Bad Request.
3. Trạng thái đơn hàng trong DB giữ nguyên là `delivered`.

## Actual results
1. Thành công: Hệ thống từ chối hủy đơn hàng ở trạng thái `delivered`.
2. Thành công: Trả về mã lỗi HTTP 400 Bad Request.
3. Thành công: Trạng thái đơn hàng giữ nguyên là `delivered`.
