# TC-FR10-08

## Requirement ID
FR10

## Module / Test type / Technique
Order State Machine / Functional / Equivalence Partitioning

## Test design source
test-design/EP-FR10.md

## Coverage
| Coverage Type | Covered IDs |
|---|---|
| Equivalence Partitioning | FR10_EP_IV01 |

## Detail
| ID | Test Objective | Token (Input) | Order ID (Input) | State Transition (Input) | Expected Output |
|---|---|---|---|---|---|
| FR10_EP_IV01 | Admin cập nhật trạng thái cho đơn hàng không tồn tại | Admin Token (TOK_A1) | Đơn không tồn tại (ORD2) | pending -> confirmed (ST1) | - Báo lỗi không tìm thấy đơn hàng (O3)<br>- Trả về HTTP 404 Not Found |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Token JWT | Admin Token hợp lệ (TOK_A1) |
| Order ID | Đơn hàng không tồn tại (ORD2) |
| Cặp chuyển đổi | pending -> confirmed (ST1) |

## Test steps
1. Gửi yêu cầu cập nhật trạng thái sang `confirmed` cho một mã đơn hàng không tồn tại (ví dụ: ID = 99999).

## Expected results
1. Hệ thống từ chối thực hiện.
2. Trả về mã lỗi HTTP 404 Not Found báo lỗi không tìm thấy đơn hàng.

## Actual results
1. Thành công: Hệ thống từ chối cập nhật trạng thái cho đơn hàng không tồn tại.
2. Thành công: Trả về mã lỗi HTTP 404 Not Found.

## Status
Passed
