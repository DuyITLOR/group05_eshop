test-design/EP-FR10.md

| Coverage Type | Covered IDs |
|---|---|

## Detail
| ID | Test Objective | State Transition (Input) | Kết quả (Output) |
|---|---|---|---|
| FR10_EP_IV01 | Admin thực hiện chuyển đổi không theo sơ đồ (`pending` -> `shipping`) | `pending` -> `shipping` (ST6) | Báo lỗi chuyển đổi không hợp lệ (O2) |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Tồn tại một đơn hàng ở trạng thái `pending` trong hệ thống.
3. Người dùng đang ở màn hình Quản lý đơn hàng (Admin).

## Test data
| Field | Value |
|---|---|
| Trạng thái hiện tại | pending |
| Trạng thái cập nhật | shipping |

## Test steps
1. Tìm đơn hàng có trạng thái `pending` trong danh sách đơn hàng.
2. Cố gắng cập nhật trạng thái của đơn hàng trực tiếp sang `shipping` (ví dụ thông qua gọi API cập nhật trạng thái với tham số `status` là `shipping`).
3. Xác nhận thực hiện.

## Expected results
1. Hệ thống từ chối cập nhật trạng thái.
2. Trả về thông báo lỗi phù hợp (ví dụ: "Invalid state transition from pending to shipping" hoặc HTTP 400 Bad Request).
3. Trạng thái đơn hàng trong DB và giao diện giữ nguyên là `pending`.
