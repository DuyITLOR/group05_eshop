# TC-D9-05

## Requirement ID
FR-20

## Module / Test type / Technique
Order Cancel (Mobile) / Functional / Equivalence Partitioning

## Test design source
test-design/EP-D9.md

## Coverage
| Coverage Type | Covered IDs |
|---|---|
| Equivalence Partitioning | FR20_EP_IV01 |

## Detail
| ID | Test Objective | Trạng thái đơn hàng hiện tại (Input) | Expected Output |
|---|---|---|---|
| FR20_EP_IV01 | Kiểm tra ẩn nút "Hủy đơn" khi đơn hàng ở trạng thái `shipping` | `shipping` (CS_M3) | Không hiển thị nút "Hủy đơn" (O_M2) |

## Preconditions
1. Người dùng đã đăng nhập tài khoản khách hàng trên ứng dụng Mobile.
2. Người dùng có đơn hàng ở trạng thái `shipping` (Đang giao).

## Test data
| Field | Value |
|---|---|
| Trạng thái đơn hàng | shipping |

## Test steps
1. Trên ứng dụng Mobile, người dùng chuyển đến tab/giao diện "Hồ sơ".
2. Cuộn xuống phần "Lịch sử đơn hàng".
3. Tìm đến đơn hàng có trạng thái "Đang giao" (`shipping`).

## Expected results
1. Nút "Hủy đơn" không được hiển thị tại đơn hàng này.

## Actual results
1. Thành công: Ẩn nút "Hủy đơn" trên ứng dụng Mobile khi đơn hàng ở trạng thái "Đang giao" (`shipping`).

## Status
Passed
