## Detail
| ID | Test Objective | JWT Token (Input) | Quyền sở hữu (Input) | Trạng thái đơn hàng hiện tại (Input) | Expected Output |
|---|---|---|---|---|---|
| FR20_EP_IV04 | Người dùng hủy đơn hàng với token không hợp lệ | Không hợp lệ (AUTH2) | Của người dùng (OWN1) | `pending` (ST1) | - Trả về HTTP 401 hoặc 403 Unauthorized (OUT4)<br>- Giữ nguyên trạng thái đơn hàng cũ, không đổi CSDL |

## Preconditions
1. Đơn hàng hợp lệ tồn tại trên hệ thống và đang ở trạng thái `pending`.

## Test data
| Field | Value |
|---|---|
| Order ID | ID của đơn hàng hợp lệ |
| Token xác thực | Token không hợp lệ hoặc đã hết hạn |

## Test steps
1. Gửi yêu cầu HTTP PUT tới API `/api/orders/{orderId}/cancel` với token xác thực không hợp lệ.

## Expected results
1. Hệ thống từ chối yêu cầu, trả về mã lỗi HTTP 401 Unauthorized hoặc 403 Forbidden.
2. Trạng thái đơn hàng trong CSDL không thay đổi.

## Actual results
1. Thành công: Hệ thống từ chối yêu cầu, trả về HTTP 403 Forbidden và `{"error":"Forbidden"}`.
2. Thành công: Trạng thái đơn hàng trong CSDL được giữ nguyên.
