## Preconditions
1. Đơn hàng hợp lệ tồn tại trên hệ thống và đang ở trạng thái `pending`.

## Test data
| Field | Value |
|---|---|
| Order ID | ID của đơn hàng hợp lệ |
| Token xác thực | Trống (không gửi kèm token) |

## Test steps
1. Gửi yêu cầu HTTP PUT tới API `/api/orders/{orderId}/cancel` mà không đính kèm header Authorization chứa token.

## Expected results
1. Hệ thống từ chối yêu cầu, trả về mã lỗi HTTP 401 Unauthorized.
2. Trạng thái đơn hàng trong CSDL không thay đổi.

## Actual results
1. Thành công: Hệ thống từ chối yêu cầu, trả về HTTP 401 Unauthorized và `{"error":"Unauthorized"}`.
2. Thành công: Trạng thái đơn hàng trong CSDL được giữ nguyên.
