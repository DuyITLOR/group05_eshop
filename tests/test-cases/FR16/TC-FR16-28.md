## Test steps
1. Tải lên file CSV trống chỉ chứa duy nhất dòng header (0 dòng dữ liệu).

2. Hiển thị thông báo lỗi cấu trúc file không hợp lệ hoặc thông báo lỗi: "Không có dữ liệu để import".
