## Detail
| ID | Test Objective | File CSV (Input) | Database State (Output) | UI Report (Output) |
|---|---|---|---|---|
| FR16_EP_IV02 | Tải lên file không có phần mở rộng (đuôi file) | File `products` (EXT3), chứa dữ liệu hợp lệ | Không có sản phẩm nào được thêm vào DB (DB2) | Báo lỗi định dạng file không hợp lệ (REP2) |

## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Người dùng đang ở màn hình Import sản phẩm từ file CSV.

## Test data
| Field | Value |
|---|---|
| File Name | products (không có phần mở rộng đuôi file) |
| File Content | Dữ liệu sản phẩm mẫu |

## Test steps
1. Chọn file `products` để tải lên hệ thống.
2. Nhấp nút "Import".

## Expected results
1. Hệ thống từ chối import và hiển thị thông báo lỗi định dạng file không hợp lệ.
2. Không có sản phẩm nào được thêm vào cơ sở dữ liệu.

## Actual results
1. Thất bại: Hệ thống không từ chối file không có phần mở rộng (đuôi file). Nếu nội dung file là định dạng CSV hợp lệ, hệ thống vẫn xử lý bình thường.
2. Thất bại: Sản phẩm được thêm thành công vào cơ sở dữ liệu.
3. Thất bại: Không hiển thị thông báo lỗi định dạng file.

## Status
Failed
