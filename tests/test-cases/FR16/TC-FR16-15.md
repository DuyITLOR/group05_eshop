## Preconditions
1. Người dùng đăng nhập hệ thống với tài khoản Admin (`role = 'admin'`).
2. Người dùng đang ở màn hình Import sản phẩm từ file CSV.

## Test data
| Field | Value |
|---|---|
| File Name | products_invalid_cat.csv |
| File Content | name,price,description,imageUrl,category_id<br>Sản phẩm A,100000,,,abc |

## Test steps
1. Chọn file `products_invalid_cat.csv` để tải lên hệ thống.
2. Nhấp nút "Import".

## Expected results
1. Hệ thống từ chối import và hiển thị thông báo lỗi: "Hàng 2: Danh mục không hợp lệ" (hoặc do không phải số nguyên).
2. Không có sản phẩm nào được thêm vào cơ sở dữ liệu.

## Actual results
1. Thất bại: Hệ thống không kiểm tra ID danh mục phải là số nguyên.
2. Thất bại: Sản phẩm có category_id = "abc" vẫn được lưu thành công vào cơ sở dữ liệu.
3. Thất bại: Giao diện báo import thành công thay vì báo lỗi danh mục không hợp lệ.
